Place your Grift Geometric Typeface webfont files in this folder.

Recommended filenames (adjust if your license provides different names):
- GriftGeometric-Variable.woff2 (preferred, 100–900)
- GriftGeometric-Regular.woff2 (weight 400)
- GriftGeometric-Bold.woff2 (weight 700)

After adding the files, the site will automatically use the font for all headings (h1–h6).
If you want to use it elsewhere, add the Tailwind class `font-grift`.
